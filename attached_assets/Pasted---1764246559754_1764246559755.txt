أريد منك الآن إجراء فحص شامل وكامل للنظام الموجود حالياً،  
وليس إنشاء مشروع جديد.

الرجاء تنفيذ المهام التالية بالتفصيل:

========================================================
## 1) فحص المشروع بالكامل (Full System Audit)
أريد منك مراجعة جميع الملفات داخل المشروع والتحقق من التالي:

- جميع الصفحات تعمل بشكل صحيح بدون أي أخطاء (Views + Controllers).
- جميع الروابط تعمل بدون 404.
- لا توجد مسارات مكسورة (Routes).
- لا توجد متغيرات غير مستخدمة.
- لا توجد دوال مكررة.
- لا توجد مكونات Blade غير مستخدمة.
- لا توجد ملفات CSS/JS غير مستخدمة.
- لا توجد مكتبات محملة بدون سبب.
- التأكد من أن كل صفحة مربوطة بالـ Controller الصحيح.
- التأكد من أن كل CRUD يعمل بشكل كامل (تنفيذ + حفظ + تعديل + حذف + عرض).
- التأكد من أن صفحات النظام الإداري والبلوكات شغّالة بدون أي أخطاء.
- التأكد من أن Sidebar مرتب ومافيه تكرار.

إذا وجدت أي أخطاء:
✓ أصلحها فوراً  
✓ واشرح السبب داخل الكود (تعليق بسيط)

========================================================
## 2) تنظيف المشروع (Project Cleanup)
قم بتنظيف كل الملفات التي لا يحتاجها النظام، وتشمل:

- حذف أي ملفات تجريبية.
- حذف أي صفحات تم استبدالها ولم تعد مستخدمة.
- إزالة أي أسطر كود غير مستخدمة.
- إزالة أي imports غير مستخدمة.
- حذف الـ Components القديمة أو غير المستعملة.
- تنظيف مجلد public من الصور أو الملفات غير الضرورية.
- تنظيف الـ storage/logs القديمة.

أريد المشروع يصبح خفيف، نظيف، مرتب، وجاهز للإنتاج Production.

========================================================
## 3) تجهيز قاعدة بيانات MySQL جاهزة لـ XAMPP
أريد منك:

### ✔ إنشاء ملف قاعدة بيانات كامل من خلال Migration  
يشمل:

- users  
- roles  
- permissions  
- role_permissions  
- user_roles  
- records  
- stations  
- ports  
- settings  
- documents (لوحدة المرسلات والكتب)  
- books_revisions (لو وجد)  
- signatures  
- activity_logs  

وتأكد أن:
- جميع العلاقات بين الجداول صحيحة (Foreign Keys).
- الكل يستخدم MySQL وليس SQLite أو PostgreSQL.
- جميع الحقول مناسبة (varchar – text – longtext – json – datetime – int).

### ✔ بعد ذلك:
إنشاء **ملف SQL جاهز للتصدير**  
(root → export database structure & seed data)

========================================================
## 4) إعداد ملف .env بشكل صحيح ومتوافق مع XAMPP + MySQL
أريدك إعداد ملف .env جاهز يحتوي على التالي:

APP_NAME="Inspection_System"
APP_ENV=production
APP_DEBUG=false
APP_URL=http://localhost

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=inspection_system
DB_USERNAME=root
DB_PASSWORD=""

SESSION_DRIVER=file
CACHE_DRIVER=file
QUEUE_CONNECTION=sync

FILESYSTEM_DISK=public

وتأكد أن:
- كل Config متوافق مع XAMPP
- كل المسارات تعمل على Windows
- لا توجد بيئة أو إعدادات خاصة بـ Linux أو Replit

========================================================
## 5) اختبار شامل للنظام بعد التنظيف
بعد إكمال التنظيف والإعداد:

- تشغيل النظام locally داخل Replit
- التأكد من أن جميع الصفحات تعمل
- التأكد من أن قاعدة البيانات تعمل
- التأكد من أن الاستيراد يعمل
- التأكد من أن رفع الملفات يعمل
- التأكد من أن البحث يعمل
- التأكد من أن نظام الكتب يعمل
- التأكد من أن الصلاحيات تعمل

أريد رسالة في النهاية:
"تم فحص النظام بالكامل، وإصلاح جميع الأخطاء، وتنظيف الملفات، وتجهيز قاعدة البيانات وملف .env، والنظام الآن جاهز للرفع على XAMPP."

========================================================

ابدأ الآن بتنفيذ جميع المهام أعلاه على النظام الحالي فقط.
