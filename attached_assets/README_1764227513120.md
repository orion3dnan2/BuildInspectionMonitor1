# نظام إدارة الرقابة والتفتيش

منصة إلكترونية شاملة لإدارة البلاغات والقيود - وزارة الداخلية، دولة الكويت

## نظرة عامة

النظام مصمم باللغة العربية ويدعم الاتجاه من اليمين لليسار (RTL). يوفر واجهة سهلة الاستخدام لإدارة السجلات والبلاغات والقيود.

---

## صفحات النظام

### 1. صفحة تسجيل الدخول (Login)
- تسجيل الدخول للمستخدمين الحاليين
- إنشاء حساب جديد للمستخدمين الجدد
- يتطلب اسم مستخدم وكلمة مرور

### 2. الصفحة الرئيسية (Home)
- عرض جميع الوحدات المتاحة في النظام
- تنقسم إلى تبويبين:
  - **نظام البلوكات**: يحتوي على لوحة التحكم، البحث، إدخال البيانات، التقارير، الإعدادات، والاستيراد
  - **النظام الإداري**: نظام إدارة الموارد البشرية

### 3. لوحة التحكم (Dashboard)
- الصفحة الرئيسية للنظام بعد تسجيل الدخول
- عرض روابط سريعة للوصول إلى: التقارير، الاستعلام، إدخال البيانات، الإعدادات، والاستيراد
- شعار وزارة الداخلية - دولة الكويت

### 4. إدخال البيانات (Data Entry)
- إضافة سجلات جديدة (بلاغات/قيود)
- تعديل السجلات الموجودة
- حذف السجلات
- عرض جدول بآخر السجلات المضافة

**البيانات المطلوبة لكل سجل:**
- رقم الصادر، الرقم العسكري
- الاسم الرباعي (الأول، الثاني، الثالث، الرابع)
- الرتبة، المحافظة، المخفر
- تاريخ الجولة، نوع الإجراء، المنافذ
- الملاحظات المدونة

### 5. البحث والاستعلام (Search)
- البحث السريع برقم السجل، الاسم، رقم الصادر، أو الرقم العسكري
- فلاتر متقدمة: المحافظة، الرتبة، المكتب، المخفر، التاريخ
- عرض تفاصيل كل سجل
- لا يمكن الحذف من هذه الصفحة (للقراءة فقط)

### 6. التقارير والإحصائيات (Reports)
- **إحصائيات سريعة:** إجمالي السجلات، سجلات اليوم، آخر شهر، آخر سنة
- **فلاتر التقارير:** الفترة الزمنية، المحافظة، المخفر، نوع الإجراء، الرتبة، البحث بالاسم
- **تقارير تفصيلية:** حسب المحافظة، الرتبة، نوع الإجراء، المخفر
- إمكانية طباعة التقارير

### 7. الإعدادات (Settings)
- **إدارة المستخدمين:** إضافة/تعديل/حذف المستخدمين وصلاحياتهم
- **إدارة المخافر:** إضافة/تعديل/حذف أسماء المخافر
- **إدارة المنافذ:** إضافة/تعديل/حذف أسماء المنافذ

### 8. الاستيراد (Import)
- استيراد البيانات من ملفات Excel (.xlsx أو .xls)
- يجب أن يحتوي الملف على الأعمدة المطلوبة بالعربية
- عرض نتائج الاستيراد (نجاح/فشل)

## المتطلبات التقنية

### التقنيات المستخدمة:
- **الواجهة الأمامية:** React + TypeScript + Vite
- **الواجهة الخلفية:** Node.js + Express
- **قاعدة البيانات:** PostgreSQL
- **التصميم:** Tailwind CSS + shadcn/ui

### متطلبات التشغيل:
- Node.js v20 أو أحدث
- قاعدة بيانات PostgreSQL
